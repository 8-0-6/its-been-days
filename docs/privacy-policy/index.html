Privacy Policy

It's Been Days Chrome Extension - Last updated: March 2026

It's Been Days helps you track tab inactivity and recover closed tabs. Privacy-first behavior is a core design principle of this extension.

What data is collected

The extension stores data locally on your device via chrome.storage.local, including:

  • Tab URLs, titles, and favicon URLs
  • Tab timestamps (opened, last visited, inactivity age)
  • Locally archived closed-tab entries (up to 500 items)
  • Local settings (notification preference and threshold)

No tab or usage data is sent to external servers.

Optional history import

The extension may request the optional history permission only if you explicitly choose to import browser history during onboarding. This import is local-only.

Accounts, payments, and third-party links

This version does not require sign-in, account creation, or subscription for core functionality. An optional support link (for example, Buy Me a Coffee) opens an external website, which is governed by that website's own privacy policy.

Permissions used

  • tabs - read tab title/URL/activity for core tab management features
  • storage - persist local metadata and settings
  • notifications - optional daily stale-tab reminder
  • activeTab - activate overlay on current tab when triggered
  • scripting - inject overlay script when requested
  • alarms - run daily inactivity checks
  • history (optional) - one-time local import only when user opts in
  • <all_urls> host permission - required to inject overlay across pages

Retention and deletion

Local extension data remains in your browser profile until you clear it in extension settings or uninstall the extension.

Policy updates

If this policy changes materially, the "Last updated" date will be revised.
